# customer-segmentation-ml
Customer Segmentation using K-Means Clustering


This project uses K-Means Clustering to group customers based on their behavior.

--> Objective
Segment customers into different groups
Understand customer behavior
Generate business insights

-->Dataset
Mall Customers dataset with:
Age
Income
Spending Score

-->ML Concepts Used
Unsupervised Learning
K-Means Clustering
Elbow Method
Data Visualization

--> Workflow
Data Loading
Data Cleaning
Feature Selection
Elbow Method
K-Means Clustering
Visualization
Business Insights
--> Results

-->Insights
Premium customers → high income & spending
Low value → low income & spending
Impulse buyers → low income, high spending
Careful → high income, low spending

-->Tech Stack
Python
Pandas
Matplotlib
Seaborn
Scikit-learn
